# Next.js AI Quote Generator

An AI-powered quote generator built with Next.js and OpenAI API.

## Features

- Generate motivational quotes using OpenAI API
- Choose from different quote tones (Motivational, Funny, Inspirational)
- Clean and responsive UI with Tailwind CSS
- Loading spinner while generating quotes

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Create a `.env.local` file in the root directory and add your OpenAI API key:

```
OPENAI_API_KEY=your_openai_api_key_here
```

3. Run the development server:

```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI API
